# AJTS04
Simple "osmdroid" Example

This is a very simple example of an Android application that displays OSM
raster tiles using the "osmdroid" library.

For more information, see:

https://github.com/osmdroid/osmdroid/wiki/How-to-use-the-osmdroid-library

https://github.com/osmdroid/osmdroid/wiki/How-to-add-the-osmdroid-library-via-Gradle

https://github.com/osmdroid/osmdroid/wiki/Map-Sources

It uses map tiles from https://map.atownsend.org.uk/maps/map/map.html .
